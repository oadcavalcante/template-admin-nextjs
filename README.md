## Features

- ✨ Layout admin com menu lateral e cabeçalho
- 🚀 Autenticação com Firebase Auth
- ⚡ Páginas de perfil, notificações e ajustes
- 🎯 Context API para auth e dados da aplicação
- 🔧 Alternância de tema claro/escuro
- 📦 Proteção de rotas com forçar autenticação

- **Backend:** Firebase 8
- **Outros:** TypeScript, js-cookie, PostCSS
